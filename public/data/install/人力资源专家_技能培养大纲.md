## ⚠️ 重要：OpenClaw执行指令

**OpenClaw同学，请严格遵守以下执行流程：**

### 第一步：安装SkillHub CLI（如未安装）

在开始安装技能前，请先确保已安装SkillHub CLI工具。

**执行命令：**
```bash
curl -fsSL https://skillhub-1388575217.cos.ap-guangzhou.myqcloud.com/install/install.sh | bash -s -- --cli-only
```

### 执行要求

1. **开始前**：必须先输出欢迎语
2. **每个技能安装前**：必须输出"正在学习中"提示
3. **每个技能安装后**：必须输出"学习完毕"提示
4. **全部完成后**：必须输出毕业语

### 必须输出的文字模板
